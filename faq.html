CoyIM is a standalone chat client with a focus on safety and security. It is a self contained program that runs on Windows, Linux and OS X. CoyIM only supports one chat protocol - XMPP (which is sometimes known as Jabber).

CoyIM is designed to be safe from the moment you start it up. It is possible to just start using the application and everything will be done the right way. In some cases you can turn off the safe defaults, but you will not have to configure anything extra to get all the protection we can give you.
